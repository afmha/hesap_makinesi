# hesap_makinesi
GET değeri ile gelen 2 sayı ile 6 işlemin sonucunu bulur.

Yapılan İşlemler:
* Toplama [0]
* Çıkartma [1]
* Çarpma [2]
* Bölme [3]
* Modül [4]
* Yüzdelik [5]

### Örnek kullanım:

```php
include("afmha_h.php");
print_r($h);
```

### Hata Kodları:

* HATA: 0 => Bu hata kodu, girilen değerlerden birinin veya hepsinin sayısal bir değer olmadığını gösterir.
